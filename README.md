# ❤️ Heart Disease Predictor using KNN

A Machine Learning project that predicts the likelihood of heart disease using the **K-Nearest Neighbors (KNN)** algorithm. This project aims to assist in early detection by analyzing key medical attributes.

---

## 🚀 Project Overview

Heart disease is one of the leading causes of death worldwide. This project uses a **supervised machine learning approach** to predict whether a person is at risk based on medical parameters such as age, cholesterol level, blood pressure, and more.

---

## 🧠 Algorithm Used

* **K-Nearest Neighbors (KNN)**

  * Simple and effective classification algorithm
  * Classifies data based on similarity (distance metric)
  * Works well with smaller datasets

---

## 📊 Features Used in Prediction

* Age
* Sex
* Chest Pain Type
* Resting Blood Pressure
* Cholesterol
* Fasting Blood Sugar
* Resting ECG
* Maximum Heart Rate Achieved
* Exercise-Induced Angina
* Oldpeak (ST depression)

## ⚙️ Installation & Setup

1. Clone the repository:

```bash

```

2. Navigate to the project folder:


3. Install dependencies:

